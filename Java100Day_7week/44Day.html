Linux 指令

Linux 命令的格式

命令名稱[命令參數][命令對象]

1.基本指令

獲取登入訊息

w / who / last/ lastb

查看自己使用Shell

ps

Shell稱為殼或殼程式，用戶和操作系統交流的翻譯，等於說是負責人與計算機之間的交流

查看命令說明和位置

whatis / which / whereis

清除螢幕上顯示的內容

clear

查看幫助文檔

man / info / --help / apropos

查看系統與主機名

uname / hostname

時間和日期

date / cal

重啟和關機

reboot / shutdown

退出登入

exit / logout

查看歷史命令

history

可以用history -c清除歷史命令

2.實用

創建/刪除空目錄

mkdir / rmdir

創建/刪除文件

touch / rm

切換/查看當前目錄

cd / pwd

cd .. 用來返回上一層

查看目錄內容

ls

-l 長格式查看文件和目錄

-a 顯示用.開頭的目錄和文件

-d 只列目錄

-S/-t 按大小/時間排序

查看文件內容

cat / tac / head / tail / more / less / rev / od

拷貝/移動文件

cp / mv

文件重新命名

rename

mv 不只有移動檔案而且可以重新命名

查找文件和查找內容

find / grep

創造/查找連結

ln / readlink

其他工具

sort 對內容排序

uniq 去掉相鄰重複內容

tr 替換指定內容為新內容

cut / paste 剪下/黏貼內容

split 拆分文件

file 判斷文件類型

wc 統計文件行數,單詞數,字節數

iconv 編碼轉換

3.用戶管理

創建/刪除用戶

useradd / userdel

d 創建用戶時指定用戶主目錄

d 創建用戶時指定所屬用戶組

創建/刪除用戶組

groupadd / groupdel

切換用戶

su

修改密碼

passwd

-d 清除用戶密碼

-e 設置用戶密碼立即過期,登入會要求修改密碼

-i 設置用戶密碼多少天過期,之後會禁用該用戶

查看/修改密碼有效期限

chage

以管理員執行命令

sudo

顯示用戶和用戶組訊息

id

給其他用戶發訊息

write / wall

4.磁碟管理

列出文件系統磁碟使用狀況

執行磁碟命令前先確定是否要這樣做，如果不清楚用法最好不要隨便使用

df

磁碟分區表操作

fdisk

磁碟分區工具

parted

格式化文件系統

mkfs

文件系統檢查

fsck

轉換/拷貝文件

dd

掛載/卸載

mount / umount

創建/激活/關閉/交換分區

mkswap / swapon / swapoff

4.磁碟管理

列出文件系統磁碟使用狀況

df
