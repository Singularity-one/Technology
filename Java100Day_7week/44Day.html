Linux 指令

Linux 命令的格式

命令名稱[命令參數][命令對象]

1.基本指令

獲取登入訊息

w / who / last/ lastb

查看自己使用Shell

ps

Shell稱為殼或殼程式，用戶和操作系統交流的翻譯，等於說是負責人與計算機之間的交流

查看命令說明和位置

whatis / which / whereis

清除螢幕上顯示的內容

clear

查看幫助文檔

man / info / --help / apropos

查看系統與主機名

uname / hostname

時間和日期

date / cal

重啟和關機

reboot / shutdown

退出登入

exit / logout

查看歷史命令

history

可以用history -c清除歷史命令

2.實用

創建/刪除空目錄

mkdir / rmdir

創建/刪除文件

touch / rm

切換/查看當前目錄

cd / pwd

cd .. 用來返回上一層

查看目錄內容

ls

-l 長格式查看文件和目錄

-a 顯示用.開頭的目錄和文件

-d 只列目錄

-S/-t 按大小/時間排序

查看文件內容

cat / tac / head / tail / more / less / rev / od

拷貝/移動文件

cp / mv

文件重新命名

rename

mv 不只有移動檔案而且可以重新命名

查找文件和查找內容

find / grep

創造/查找連結

ln / readlink

其他工具

sort 對內容排序

uniq 去掉相鄰重複內容

tr 替換指定內容為新內容

cut / paste 剪下/黏貼內容

split 拆分文件

file 判斷文件類型

wc 統計文件行數,單詞數,字節數

iconv 編碼轉換

3.用戶管理

創建/刪除用戶

useradd / userdel

d 創建用戶時指定用戶主目錄

d 創建用戶時指定所屬用戶組

創建/刪除用戶組

groupadd / groupdel

切換用戶

su

修改密碼

passwd

-d 清除用戶密碼

-e 設置用戶密碼立即過期,登入會要求修改密碼

-i 設置用戶密碼多少天過期,之後會禁用該用戶

查看/修改密碼有效期限

chage

以管理員執行命令

sudo

顯示用戶和用戶組訊息

id

給其他用戶發訊息

write / wall

4.磁碟管理

執行磁碟命令前先確定是否要這樣做，如果不清楚用法最好不要隨便使用

列出文件系統磁碟使用狀況

df

磁碟分區表操作

fdisk

磁碟分區工具

parted

格式化文件系統

mkfs

文件系統檢查

fsck

轉換/拷貝文件

dd

掛載/卸載

mount / umount

創建/激活/關閉/交換分區

mkswap / swapon / swapoff

5.配置服務

啟動防火牆服務

systemctl start firewalld

關閉防火牆服務

systemctl stop firewalld

重啟防火牆服務

systemctl restart firewalld

查看防火牆狀態

systemctl status firewalld

6.網路訪問與管理

安全遠程連結

ssh

透過網路獲取資源

wget

-b 後臺下載模式

-O 下載到指定目錄

-r 遞迴下載

發送接收 email

mail

網路配置工具

ip 新版本

ifconfig 新版本

網路可達性檢查

ping

顯示管理路由表

route

查看網路服務和端口

netstat / ss

網路監聽抓包

tcpdump

安全文件拷貝

scp

文件同步工具

rsync

安全文件傳輸

sftp

7.進程管理

查看進程

ps

顯示進程狀態樹

pstree

查找指定條件匹配的進程

pgrep

通過進程號終止進程

kill

通過進程名終止進程

killall / pkill

查看後台進程

jobs

將進程置於後台運行

Ctrl + z 將進程置於後台運行

讓進程在後台繼續運行

bg

讓後台進程置於前台

fg

置於前台的進程可以用Ctrl +c 來終止

調整程式/進程的優先順序

nice / renice

用戶登出後進程繼續工作

nohup

跟蹤進程系統調用狀況

strace

查看當前運行級別

runlevel

實時監控進程占用狀況

top

-c 顯示進程整個路徑

-d 指定兩次刷屏之間的間隔時間(秒)

-i 不顯示閒置進程

-p 顯示指定進程訊息

8.系統診斷

系統啟動異常診斷

dmesg

查看系統活動訊息

sar

-A 顯示所有設備(CPU,內存,磁碟)運行的情況

-u 顯示所有CPU負載情況

-d 顯示所有磁碟使用情況

-r 顯示所有內存使用情況

-n 顯示網路運行狀態

查看內存使用情況

free

虛擬內存統計

vmstat

CPU訊息統計

mpstat

查看進程使用內存情況

pmap

報告設備CPU和I/O統計訊息

iostat

顯示所有PCI設備

ispci

顯示所有PCI設備

ispci

顯示進程間通訊設備的狀態

ipcs

9.常用快捷鍵

自動補全目錄或全路徑

tab

快捷鍵 功能說明
tab 自動補全目錄或全路徑
Ctrl+a 將指標移到命令行首
Ctrl+e 將指標移到命令行尾
Ctrl+f 將指標向右移動一個字符
Ctrl+b 將指標向左移動一個字符
Ctrl+b 將指標向左移動一個字符
Ctrl+b 將指標向左移動一個字符
Ctrl+k 剪切指標到行尾的字符
Ctrl+u 剪切指標到行首的字符
Ctrl+w 剪切指標前面一個單詞
Ctrl+y 複製命名剪切的內容
Ctrl+c 中斷正在執行的任務
Ctrl+h 刪除指標前面的一個字符
Ctrl+d 退出當前命令
Ctrl+r 搜索歷史命令
Ctrl+g 退出歷史命令搜索
Ctrl+s 鎖定終端暫時無法輸入內容
Ctrl+q 退出終端鎖定
Ctrl+z 將正在終端執行的任務停下放到後台
!! 執行上一條命令
!數字 執行對應數字的歷史命令
Esc+b 移動到當前單詞的開頭
Esc+f 移動到當前單詞的尾
