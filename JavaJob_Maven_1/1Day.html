第1天

Maven環境建置

建立第一個maven專案

前言

主要參考 Maven in 5 Minutes

希望在工作上對於使用的Maven 可以多了解一些

第一步 Maven環境建置

1

在Maven 官網 下載 Maven

開始功能表搜尋PowerShell 和 右鍵選以系統管理員身分執行

2

使用 PowerShell 輸入指令

官方連結

如果是比較舊的版本

按照網頁依序輸入

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

wsl --set-default-version 2

第二步 安裝 Docker Desktop

1

Google 搜尋Docker

點選 Get Started

選擇 windows 環境下安裝下載後執行

2

安裝完成後可以檢查輸入指令 docker --version

第三步 安裝Jenkins

1

使用 PowerShell 輸入指令

按照網頁依序輸入

docker pull jenkins/Jenkins

2

docker images 確認 有jenkins

3

輸入 docker run -p 8080:8080 -p 50000:50000 jenkins/jenkins

留意以下字串是密碼

4

打開瀏覽器輸入http://localhost:8080/ 會看到這畫面

輸入 剛才的密碼

點選左邊社群推薦安裝

設定帳號和密碼

5

重啟登入輸入帳號和密碼

6

起始畫面

第四步 Jenkins Hello World

1

新增作業>創建第一個pipeline

2 輸入以下內容

輸入以下內容

pipeline{
    agent any
 
    stages{
       stage('Build'){
          steps{
             echo 'Hello world'
          }
       }
    }
 }

3

察看結果
